O tamanho da fonte na visualização é determinado pelo tamanho da fonte especificado ao criar os elementos de texto no canvas, e não pelo tamanho
da fonte especificado no código ZPL. Você pode tentar modificar a função update_preview para extrair o tamanho da fonte do código ZPL e usá-lo ao
criar os elementos de texto no canvas. Por exemplo, você pode modificar a função update_preview para ficar assim:
def update_preview():
# Obtém o código ZPL do campo de texto
    zpl_code = text_input.get("1.0", tk.END)
# Analisa o código ZPL para extrair informações sobre os elementos
    elements = []
for match in re.finditer(r'\^FO(\d+),(\d+)\^A[^\^]+,(\d+),(\d+)\^FD([^\^]+)\^FS', zpl_code):
        x, y, font_height, font_width, text = match.groups()
        elements.append(('text', int(x), int(y), int(font_height), int(font_width), text))
for match in re.finditer(r'\^FO(\d+),(\d+)\^BY[^\^]+\^B[^\^]+\^FD([^\^]+)\^FS', zpl_code):
        x, y, data = match.groups()
        elements.append(('barcode', int(x), int(y), data))
# Limpa o canvas e o dicionário de códigos de barras
    canvas.delete("all")
    canvas.barcode_data = {}
# Adiciona um retângulo para delimitar a área de visualização da etiqueta
    x_offset = 11
    y_offset = 10
    canvas.create_rectangle(10 + x_offset, 10 + y_offset, 590 + x_offset, 590 + y_offset)
# Cria objetos gráficos interativos para representar os elementos
for element in elements:
if element[0] == 'text':
            _, x, y, font_height, font_width, text = element
            item = canvas.create_text(x + 50 + x_offset, y + 50 + y_offset, text=text, anchor=tk.NW, font=("Calibri Bold", font_height))
            canvas.tag_bind(item, "<ButtonPress-1>", on_element_press)
            canvas.tag_bind(item, "<B1-Motion>", on_element_drag)
elif element[0] == 'barcode':
            _, x, y, data = element
            writer = ImageWriter()
            writer.dpi = 300
            writer.module_height = 5.0
            upca = barcode.get('upca', data, writer=writer)
            image = upca.render()
            image.thumbnail((200, 200))
            photo = ImageTk.PhotoImage(image)
            item = canvas.create_image(x + 50 + x_offset, y + 50 + y_offset, image=photo, anchor=tk.NW)
            canvas.barcode_data[item] = data
            canvas.tag_bind(item, "<ButtonPress-1>", on_element_press)
            canvas.tag_bind(item, "<B1-Motion>", on_element_drag)
            canvas.image = photo
Isso deve extrair o tamanho da fonte do código ZPL e usá-lo ao criar os elementos de texto no canvas. No entanto, observe que o tamanho da fonte
especificado no código ZPL pode não corresponder diretamente ao tamanho da fonte em pixels na visualização. Você pode precisar ajustar o tamanho
da fonte ao criar os elementos de texto no canvas para obter o resultado desejado. Isso ajuda?
